AI 노트필기

나은주

이 프로젝트를 통해 AI와 더 친숙해지고, 저의 개발 경험에 의미 있는 이력을 더하고 싶습니다. AI 기술을 활용하여 실제 사용자에게 가치를 제공하는 프로젝트를 완성하는 것이 목표입니다.

프로젝트: AI 노트필기

AI와 함께 공부하는 학생들이 늘어나는 트렌드에 맞춰, 효율적인 노트필기와 학습을 동시에 지원하는 AI 서비스를 개발하였습니다.

프로젝트 개요

  • 기 간: 2025/11/01 ~ 2025/11/17
  • 배 경: AI 기반 학습 보조 도구 수요 증가

사용 기술

백엔드: Spring Boot Python FastAPI
프론트엔드: React JavaScript HTML5
데이터베이스: MySQL
기타 기술: Figma Git

핵심 기술

1. AI 핵심 요약 기능

DB와 FastAPI를 이용해 사용자가 자료(pdf, word, ppt...)를 추가하면 AI가 요약, 해석, 설명을 제공합니다.

2. 관련 유튜브 영상 추천

AI가 업로드된 자료의 맥락을 파악하여, 학습에 도움이 되는 관련 유튜브 영상을 찾아 사용자에게 추천합니다.

3. 노트필기 형식 AI 모델

"노트필기 형식"이라는 특정 프롬포트에 반응하여, AI가 자료를 체계적인 노트필기 형태로 재구성하여 대답하도록 모델을 학습/조립했습니다.

문제 정의

"과연 기본적인 학습 과정을 AI가 어느 정도까지 의미 있게 대체하고 보조할 수 있는가?"

해결 과정

  • 모델 학습 방식: 단순 채팅이 아닌, 사용자가 제공한 소스를 기반으로 응답을 생성하도록 GPT API를 활용하여 모델을 강화했습니다.
  • Notion 방식 벤치마킹: 중요한 부분에 색상을 칠하고 제목을 굵게 표시하는 등, 가독성 높은 UI/UX를 벤치마킹하여 유연한 노트필기 방식을 구현했습니다.

성과 및 결과

글씨체와 정렬을 사용자가 커스터마이징할 수 있는 노트필기 UI를 구성하였으며, AI가 자료를 읽고 요약/해석해주는 핵심 기능을 성공적으로 확장하여 구현하였습니다.

배운 점

"고작 API 키를 가져와 간단한 AI 채팅 기능을 구현하는 과정에서도 수많은 우여곡절을 겪었습니다. 이를 통해 프로젝트 완성에 얼마나 많은 기술적 노력이 필요한지, 그리고 저에게 아직 더 많은 학습과 경험이 필요하다는 것을 깊이 깨닫게 되었습니다."
